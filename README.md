# 🌙 Alibaba Dark Mode Extension

A lightweight Chrome extension that enables a consistent dark theme on Alibaba.com, specifically optimized for the Messenger and chat interfaces.

---

## ✨ Features

- **Global Dark Mode:** Clean, high-contrast dark theme for better readability and reduced eye strain.
- **SVG Color Correction:** Automatically adjusts icon colors to fit the dark theme.
- **Dynamic Content Support:** Uses `MutationObserver` to style newly loaded elements instantly.
- **One-Click Toggle:** Switch between Dark and Light mode using a floating button.
- **Zero Dependencies:** Built with pure JavaScript and CSS for maximum performance.

## 📦 Installation

### 1. Download the Project

Clone the repository:

```bash
git clone https://github.com/Alhomaidi1/Alibaba-Dark-Mode.git
````

Or download it as a ZIP and extract it.

---

### 2. Load the Extension in Chrome

1. Open Chrome and go to:

   ```
   chrome://extensions/
   ```
2. Enable **Developer mode** (top right)
3. Click **Load unpacked**
4. Select the project folder

---

## 🚀 Usage

1. Open [https://www.alibaba.com](https://www.alibaba.com)
2. The extension will activate automatically
3. Use the button in the bottom-right corner:

   * ☀️ Light Mode
   * 🌙 Dark Mode

---

## 🛠️ Technical Details

* **Manifest V3** compliant
* Targets specific Alibaba UI containers for better styling control
* Uses `MutationObserver` for real-time DOM updates
* Caches original SVG colors for accurate restoration
